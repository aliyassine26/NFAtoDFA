DFA Characteristics

  • DFA: Deterministic Finite Automata
  • A DFA is a 5-tuple (Q,ε,𝛿,q,F)
  • Q is a finite set called the states
  • ε is a finite set called the alphabet
  • 𝛿:Q×ε→p(Q) is the transition function
  • q∈Q is the start state
  • F⊆Q is the set of accept/final state

NFA Characteristics

  • NFA: Deterministic Finite Automata
  • A NFA is a 5-tuple (Q,ε,𝛿,q,F)
  • For any set Q,p(Q) is the collection of all subset of Q
  • q∈Q is the start state
  • F⊆Q is the set of accept/final state
